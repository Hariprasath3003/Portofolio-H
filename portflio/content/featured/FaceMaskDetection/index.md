A Python-based Smart Car Parking System using OpenCV for real-time license plate recognition. The system detects vehicle plates, fetches owner details, auto-calculates parking charges, and sends SMS notifications with an online payment link. Integrated with Firebase for smooth data handling and automated parking flow.
